- 🎨 **Modern UI** - Clean and professional interface with gradient backgrounds

- 📦 **Modular Structure** - Easy to customize and extend

## How to Customize

### Update Personal Information

1. Edit `index.html` to update:
   - Your name and title
   - About me section
   - Projects details
   - Contact information

2. Edit `styles.css` to customize:
   - Colors (see `:root` CSS variables)
   - Fonts
   - Spacing
   - Layout

3. Edit `script.js` to add:
   - Additional interactivity
   - Form handling
   - Analytics

### Change Colors

Update the color variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;  /* Main color */
    --secondary-color: #ec4899; /* Accent color */
    --text-color: #1f2937;
    --bg-color: #ffffff;
    --light-bg: #f9fafb;
}
```
